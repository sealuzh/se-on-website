Annotations (1)

  • comment "The acronym SEON stands for Software Evolution ONtologies and represents our attempt to formally describe knowledge from the domain of software evolution analysis and mining software repositories. SEON consists of multiple ontologies, for example to describe stakeholders, activities, artifacts, and the relations among all of them. The ultimate goal is to facilitate the implementation of tools that help software engineers to manage software systems over their entire life-cycle. The SEON-ALL ontology, found at http://se-on.org/ontologies/all.owl, does not define any concepts - instead, it imports all the other SEON ontologies for the sake of documentation." ()
