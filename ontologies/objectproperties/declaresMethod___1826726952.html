Annotations (1)

  • comment "A class or interface can declare zero, one, or several methods. If an interface declares a method then the method has to be abstract, i.e., it cannot have a body. Note that a class can inherited additional methods from its superclass(es). This property is only applicable to method that are explicitely declared in the given class. The inverse property is isDeclaredMethodOf." ()
