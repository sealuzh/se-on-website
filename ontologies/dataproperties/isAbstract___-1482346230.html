Data Property: isAbstract

http://se-on.org/ontologies/domain-specific/2012/02/code.owl#isAbstract

Annotations (1)

  • comment "Classes and methods can be declared abstract. If a class is abstract, it cannot have any instances. However, it can be extended by a non-abstract subclass. If a method is declared abstract then it cannot have a body. Instead, an implementation needs to be provided by a subclass (or the subclass needs to be abstract itself). Annotations and interfaces are implicitly abstract and, accordingly, their methods too." ()
